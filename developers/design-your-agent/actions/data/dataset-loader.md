# 数据集加载器

`数据集加载器` 动作允许您从代理的数据集中加载数据。数据可以在后续动作中使用。

## 用法

* 将 `数据集加载器` 动作添加到您的代理中，并选择要加载的数据集。

* 在下游动作中使用数据集的名称引用这些数据，例如：`{{DATA_2}}`。
